# sword
Sword 是一款单体架构的内存数据库，基于java实现，希望它能像一把绝世好剑一样无坚不摧。
* 列存储
* 对象存储
* 时序数据存储
* 堆外存储
* 可持久化
* 利用原子运行器保证数据的一致性
